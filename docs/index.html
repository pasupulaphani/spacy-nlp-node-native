spacy-nlp-node-native[wip]

Spacyio nlp for nodejs with native bindings

Prerequisites

  • spaCy installed

Getting started

    npm install spacy-nlp-node-native

Usage

    const spacyNlp = require("spacy-nlp-node-native");

    spacyNlp.parse("test me", {});
